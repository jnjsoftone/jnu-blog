### 목적
- 웹브라우저에서 동작하는 게임을 제작한다.
- 서버 및 데이터베이스 기능을 제공하는 Firebase의 사용법을 익힌다. 
- React 프레임워크를 사용하여 프론트엔드를 구성한다.

### Tech Stack

> HTML, CSS, Javascript, React.js, Firebase, MUI
