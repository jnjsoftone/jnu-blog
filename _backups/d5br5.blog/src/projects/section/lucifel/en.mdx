### Background

- ‘Lucifel’, a grand mansion butler cafe located in Yeonnam-dong
- Previously accepted [reservations via Naver](https://map.naver.com/p/search/%EB%A3%A8%EC%B9%98%ED%8E%A0/place/1948793457?c=15.00,0,0,0,dh&placePath=/home)
  - Only Naver Pay was accepted, making it difficult for foreign tourists to make reservations
- Built an online reservation and payment system to allow international customers to make reservations

### Tech Stack

- Next.js, Supabase, Shadcn/ui, Tailwind CSS

### Role

> Designed the database structure and reservation system
Developed both the client and admin pages (Full-Stack)

### Key Features
 
#### Client
	- Check available dates and times
	- Enter personal information and confirm reservation after payment (Paypal)
	- Send a confirmation email with reservation details (Mailersend)
	- View reservation information using the order number
	- Multilingual support (Korean, English, Japanese, Chinese)

![](/projects/lucifel/lucifel_client.jpg)


#### Admin Page
	- Open and edit available reservation schedules
	- View the list of reservations and detailed information (monthly, daily)
	- Add and edit reservation products and announcements
